# Task Manager API

Backend for a task-management  API app using Node.js, Express and PostgreSQL.

## Features
- Assign tasks to users (foreign keys)
- Password hashing (bcryptjs)
- Tasks with statuses (`todo`, `in-progress`, `done`)
- Seed script with sample data
- Pagination + filtering
- Rate limiting
- Input validation
- Tests (Jest + Supertest)
- JWT authentication (jsonwebtoken)
